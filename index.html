구글링하자

각 칸에 값을 넣으면 구글 검색문장을 자동으로 만들어 줍니다.
연산자를 외울 필요 없이, 원하는 조건만 입력하고 “구글에서 검색하기”를 누르면 됩니다.

1. 기본 검색 설정
따옴표 안 문장이 정확히 포함된 결과만 찾고 싶을 때
쉼표(,)로 여러 개 입력 → (고양이 OR 강아지) 형태로 묶어서 검색
쉼표(,)로 여러 개 입력 → (경제 AND 정책) 형태로 묶어서 검색
쉼표(,)로 여러 개 입력 → +공식 +PDF 처럼 강제 포함
* 위치에 어떤 단어든 들어가도 되게 검색 (예: 서울 <아무말> 맛집)
쉼표(,)로 여러 개 입력 → -블로그 -카페 -광고
2. 사이트 / 제목 / URL / 파일
3. 날짜 / 출처 / 위치 / 숫자 범위
예: camera $100..$300 처럼 쓸 때 숫자 범위를 넣어두고, 검색어에 함께 사용
4. 기타 연산자 / 기능
두 단어가 가까이 있는 결과 찾기: word1 AROUND(n) word2
5. 특수 검색 (weather / stocks / map / movie / info)
생성된 검색문장
아직 아무 것도 입력하지 않았습니다.

이 페이지에는 쿠팡 파트너스 광고가 포함되어 있습니다.
해당 링크를 통해 구매가 이루어질 경우, 사이트 운영자는 일정 수수료를 받을 수 있습니다.
